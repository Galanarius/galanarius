Class: NPC

NPC(x, y, loc, ldr, t)

new NPC(x, y, loc, ldr, t)

Creates an NPC profile and saves it to a json file in the npcs folder at root level of the project.
Parameters:
Name Type Description
x Number The x-coordinate where the NPC is to reside upon creation.
y Number The y-coordinate where the NPC is to reside upon creation.
loc String The name of the planet or planetoid the NPC is to reside at upon creation.
ldr String The name of the NPC leading this NPC (pass 'self' to make it its own leader).
t Number The tier (1,2,3, or 4) of the NPC, will be randomly selected if not specified.
